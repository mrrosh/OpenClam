OpenClam (v0.7)
Search clam shells in inventory and open them after looting

note
Loot window closes when moving ... so stand still a second after looting a clam
It also should work after receiving clams from bank, mail, trades (but this hasent been tested yet)

slash commands
/clam enable ................. enable addon
/clam disable ................ disable addon
/clam status ................. status of addon
/clam count .................. count clams in inventory
/clam open ................... open clam manualy
/clam version ................ addon version
